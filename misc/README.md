## Miscellaneous Scripts for the Minecraft Server

Export these scripts to your `$PATH` by adding the following line to your `~/.zshrc` or `~/.bashrc`.

```sh
export PATH="$HOME/Documents/Repositories/server/misc:$PATH"
```
